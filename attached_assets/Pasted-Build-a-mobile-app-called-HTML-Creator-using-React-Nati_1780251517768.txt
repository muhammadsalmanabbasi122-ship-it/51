Build a mobile app called "HTML Creator" using React Native (Expo). The app has the following features:

🔐 Authentication
Login and Register screens with email & password
Password visibility toggle
Local storage se session save ho (AsyncStorage)
App open hone par auto-login
Purple gradient background, Inter font, modern card UI
🏠 Home Screen
User ka greeting dikhao (e.g. "Welcome, Ali!")
"Online" green status badge
Quick access cards: AI Generator, Upload HTML, Community, Profile
"Get Started" tips section
Smooth fade-in animations
🤖 AI Generator Screen
Chat-style interface
User HTML describe karta hai (e.g. "make a landing page for a bakery")
AI (Kimi K2 model) HTML code generate karta hai via backend API /api/ai/generate
One-tap copy button for generated code
Example prompt suggestions
Haptic feedback on interactions
📤 Upload Screen
WebView mein https://web.chandtricker.qzz.io open ho
Loading spinner dikhao jab page load ho raha ho
Error message agar page load na ho
💬 Community Screen
Real-time group chat room
Har 3 second mein messages fetch ho (polling)
Online users list with initials avatars
Backend APIs: GET /api/chat/messages, POST /api/chat/messages, GET /api/chat/online
👤 Profile Screen
User initials avatar (e.g. "AK" for Ali Khan)
Join date, upload count, plan (Free)
Last 10 uploads ki list (filename, size, date)
Logout button
🎨 Design System
Primary color: #6D28D9 (purple)
Accent: #0EA5E9 (sky blue)
Background light: #F5F3FF (lavender)
Background dark: #09090F (near black)
Font: Inter (Regular, Medium, SemiBold, Bold)
Border radius: 14px
Dark mode support automatic
Icons: @expo/vector-icons (Feather)
LinearGradient backgrounds
Haptic feedback on buttons
FadeInDown/FadeInUp animations (react-native-reanimated)
🗂️ Navigation
Expo Router file-based routing
Bottom tab bar: Home, AI, Upload, Community, Profile
Auth stack: Login → Register
Unauthenticated users auto-redirect to Login
⚙️ Tech Stack
- Expo SDK 54
- React Native
- Expo Router (tab + stack navigation)
- AsyncStorage (local user data)
- TanStack React Query
- react-native-webview
- expo-linear-gradient
- expo-haptics
- expo-clipboard
- Inter fonts (@expo-google-fonts/inter)
- TypeScript
- Zod (validation)
